# Personal AI Assistant

Personal AI Assistant is a mobile application designed to help users capture, store, and retrieve personal information using voice and natural language.

The app acts as a personal memory system powered by artificial intelligence, allowing users to record information and later query it conversationally.

---

# Core Idea

Users can speak naturally to the app to save information about their life, activities, expenses, objects, and events.

Later they can ask questions in natural language and the AI assistant will retrieve relevant information from their personal knowledge base.

Examples:

Save information:
- "I bought shoes in Rome and paid 25 francs."
- "I spent 150 francs on the motorcycle: chain, oil and maintenance."
- "I lent money to Marco."
- "I stored four boxes of peas in the cellar."

Ask questions:
- "How much did I spend on the motorcycle last year?"
- "What did I buy in Rome?"
- "Who owes me money?"
- "What did I store in the cellar?"

The assistant searches the user's stored data and returns intelligent answers.

Memory persistence always requires explicit user confirmation (`Confirm / Modify / Cancel`).

---

# Main Features

### Voice Memory Capture
Users can quickly record information using voice input.

### Receipt Photo Capture
Users can attach only receipt photos (camera or gallery). The app runs OCR and proposes structured memory data before confirmation.

### AI Question Interface
Users can ask questions about their stored information in natural language.

### Personal Knowledge Base
All saved information becomes part of the user's searchable personal memory.

### AI Insights Dashboard
The app provides useful insights such as:
- spending summaries
- activity statistics
- frequently referenced information
- suggestions generated by AI

---

# Target Users

The application is designed for:

Private individuals who want to organize their personal information and memory.

Professionals who need to keep track of work-related notes, expenses, or activities.

Small teams/professionals are a future expansion path, while MVP remains personal-memory focused.
