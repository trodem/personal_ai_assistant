# Non Goals (MVP)

The first version of Personal AI Assistant focuses on core memory capture and AI retrieval.

The following features are intentionally not included in the initial version.

---

# Advanced Automation

The assistant will not perform complex automated actions such as:

- booking services
- controlling smart home devices

The assistant focuses on memory and information retrieval.

---

# Multi-user collaboration

The first version will not support:

- shared memories between users
- team workspaces

Each account manages its own private memory system.

---

# Offline AI processing

All AI processing will run through cloud services.

Local AI models on the device are not part of the MVP.

---

# Complex workflow systems

The application will not include advanced workflow builders or automation pipelines.

The focus remains on simple natural language interaction.

---

# Enterprise integrations

The first version will not include integrations with:

- CRM systems
- ERP platforms
- enterprise databases

These may be added in future versions.

---

# Custom AI models

The first version will use external AI providers.

Training custom models is not part of the MVP.

---

# Full document knowledge base

While files can be attached to memories, the first version will not include full document indexing systems for large document collections.

Attachments are primarily contextual references for individual memories.
